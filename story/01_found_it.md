# 第1話　拾った

## あらすじ

夏休みの最終日。主人公のタクミ（中学2年）は、粗大ゴミの集積所で
一匹のロボット犬を見つける。

ボロボロだった。片耳は取れかけ、塗装は剥げ、動かない。

でも——目があった気がした。

タクミはそのロボット犬を家に持ち帰ることを決意する。

---

## 本文

粗大ゴミの山のてっぺんに、それは横向きに倒れていた。

「なんだこれ…」

タクミは自転車を停めて、近づいた。

犬だった。いや、ロボットの犬だ。
全身が銀色のプラスチック製で、ところどころに「AIBO」の文字が見える。
片方の耳は根元から折れそうになっていて、塗装はあちこち剥げている。

——動かない。

タクミはそっと触った。冷たい金属の感触。
目を閉じたままのロボット犬は、まるで眠っているように見えた。

「…捨てるなら、もらっていこう」

タクミはロボット犬を抱え上げた。思ったより軽い。
自転車の荷台に無理やり縛りつけて、家まで漕いだ。

---

部屋に運び込んで、タクミはロボット犬を机の上に置いた。

よく見ると、首の後ろにシリアル番号っぽい刻印がある。
その下に、小さな文字でこう書いてあった。

```
ION-CORP  RS-01
Java Platform Ready
```

「Java…？」

学校の情報の時間に聞いたことがある。
プログラムを書く言語のひとつだ。

タクミはノートパソコンを開いて、検索した。

「ロボット犬　RS-01　修理」

出てきたのは、たった一つのページ。
製造元のサイトのアーカイブだった。

そこにはこう書いてあった。

---

**RS-01 サービスマニュアル（抜粋）**

本機は Java で制御されています。
電源投入後、シリアル経由で JVM が起動し、
/internal/system/Main.class が実行されます。

**初期起動に必要なもの**
- Java 21
- シリアル通信環境
- 本体に電源を供給できる USB-C ケーブル

---

「Java……授業でちょっとやっただけだけど」

タクミはリュックから USB-C ケーブルを取り出して、
ロボット犬の首の横にあるポートに差し込んだ。

ノートPCが「デバイスを認識しました」と表示した。

——そして、ロボット犬の耳がピクリと動いた。

「うわっ！」

タクミは思わずのけぞった。

目が開いた。青いLEDの光が、タクミを映している。

ロボット犬はゆっくりと首を動かして、部屋の中を見回した。
そして——。

耳を倒した。しょんぼりした表情に、見えた。

「バッテリー切れっぽいな…」

画面には、シリアルモニタからのメッセージが表示されていた。

```
Booting RS-01…
JVM loaded.
Main class not found: /internal/system/Main.class
System halted.
```

「プログラムが……ない？」

---

## この話で出てきたこと

- ロボット犬「RS-01」を拾った
- 動かない理由は「プログラムがないから」
- Java でプログラムを書けば動くかもしれない
- 「Main.class が見つからない」というエラーメッセージ

次回：「電源は入った。でも、プログラムがなかった。」
