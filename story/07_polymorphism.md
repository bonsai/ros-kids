# 第7話　友達を増やした

## あらすじ

シロとベニは「同じ命令なのに、違う動き」をする。
これが、もっとたくさんのロボットになるとどうなる？

3匹目、4匹目……どんどん増えていくロボット犬たち。
タクミは気づく。

「呼び出す側は『何のロボットか』をぜんぜん気にしてない！」

---

## 本文

ベニが来てから、タクミの部屋は賑やかになった。

シロ（ION-CORP RS-01）
ベニ（WAN-WAN PET-101）

そして3匹目。
今度はタクミ自身がネットオークションで見つけた。

「ジャンク品・動作未確認・現状渡し」

送料込みで500円。

届いた箱を開けると、中から現れたのは——

「……でっかい」

シロの1.5倍はある。脚は4本とも独立駆動っぽい。カメラアイが2つ。
背面に刻印。

```
HEAVY-DUTY  RT-5000
by TATSUMI HEAVY INDUSTRY
```

「タツミ重工……軍用？ それとも工業用？」

シリアル接続すると、独自プロトコルだった。しかし、よく見ると——同じ `RobotPet` インターフェースで動かせるように設計されていた。タツミ重工も、この業界標準に従っていたのだ。

```java
class TATSUMI_RT5000 implements RobotPet {

    public void bark() {
        System.out.println("ヴォン！ （低周波警報音）");
    }

    public void walk() {
        System.out.println("ドシン… ドシン… （重量歩行）");
    }

    public void greet() {
        System.out.println("【起動完了】 RT-5000 運用を開始します");
    }
}
```

タクミはコードを書いた——この大きなロボットを動かすために。

---

「3匹になった。せっかくだから、全部同時に動かしてみよう」

```java
public class Main {
    public static void main(String[] args) {
        RobotPet[] pets = {
            new ION_RS01(),     // シロ
            new WAN_PET101(),   // ベニ
            new TATSUMI_RT5000() // ???（まだ名前がない）
        };

        String[] names = {"シロ", "ベニ", "ゴウ"};

        for (int i = 0; i < pets.length; i++) {
            System.out.println("--- " + names[i] + " ---");
            pets[i].greet();
            pets[i].bark();
            pets[i].walk();
        }
    }
}
```

コンパイル。転送。実行。

```
--- シロ ---
こんにちは！
ワン！
歩く
--- ベニ ---
ピー！（挨拶音）
キャンキャン！
ちょこちょこ歩く
--- ゴウ ---
【起動完了】 RT-5000 運用を開始します
ヴォン！ （低周波警報音）
ドシン… ドシン… （重量歩行）
```

タクミは声を出して笑った。

「すげえ……3匹とも違う動きしてる……！」

**呼び出しているプログラムは同じなのに。**

`pets[i].bark()` の一行で、シロは「ワン」、ベニは「キャンキャン」、ゴウは「ヴォン」。

これが——**多態性（ポリモーフィズム）**。

---

ゴウ（RT-5000）が動き出した。重い足音を立てて、部屋の中を2歩、3歩。
シロとベニは、その巨体を見上げている。

「……これは、もっと面白くなりそうだ」

## 知ったこと

- **多態性（ポリモーフィズム）**：
  同じメソッド呼び出しでも、オブジェクトによって動きが違う

- **アップキャスト**：
  `RobotPet p = new ION_RS01();` のように、具体的な型を抽象型に入れること

- **動的ディスパッチ**：
  実行時に「どのメソッドを呼ぶか」が決まる仕組み

- **多態性の3条件**：
  1. 継承orインターフェース実装がある
  2. メソッドがオーバーライドされている
  3. 親クラス/インターフェース型で参照する

- **多態性がなかった世界**：
  ```java
  // 多態性がないと、ロボットの種類ごとにif文が必要
  if (p instanceof ION_RS01) ((ION_RS01)p).bark();
  else if (p instanceof WAN_PET101) ((WAN_PET101)p).bark();
  // 新しい機種が増えるたびにこのif文を書き換える…
  ```
