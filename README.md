# ロボット犬で学ぶ Java & ロボティクス

**ロボティクス × Java × OOP を、読み切り短編で楽しく学ぼう。**

ロボット犬とリアル犬を題材に、クラス・インターフェース・継承・多態性・
ROS 風ノード通信・頭脳（Brain）差し替えまでを
**GitHub Pages 上で動く Java プレイグラウンド** で試せる教材です。

## 構成

| フォルダ | 内容 |
|----------|------|
| `docs/` | 読み切り教材（Markdown） |
| `playground/` | TeaVM Java プレイグラウンド（ブラウザで動く） |
| `src/` | TeaVM 用 Java ソース |

## プレイグラウンド

GitHub Pages を有効にするとここで動きます：

```
https://あなたのユーザー名.github.io/ros-kids/playground/
```
