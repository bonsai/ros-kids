# 第1話：ロボット犬が研究所に届いた！

きみはロボット研究所の新人エンジニア。
今日は「ロボット犬」と「本物の犬」を、どちらも"犬"として扱えるようにする仕事がきた。

Java では、こういうとき **クラス** と **インターフェース** を使う。

## 今日の学び
- クラスは「設計図」
- オブジェクトは「実体」
- ロボット犬もリアル犬も、Java では同じように扱える

## サンプルコード

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("ロボット犬が研究所に到着！");
    }
}
```
