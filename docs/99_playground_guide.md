# プレイグラウンドガイド

GitHub Pages 上で動く Java プレイグラウンドの使い方。

## 開き方

```
https://あなたのユーザー名.github.io/ros-kids/playground/
```

## 使い方

1. 左のテキストエリアに Java コードを書く
2. 「実行」ボタンを押す
3. 右側に実行結果が表示される

## サンプルコード

ページを開くと最初から ROS 風 pub/sub のコードが表示されています。
コードを自由に書き換えて試してみよう。

## 自分だけのロボット犬を作ろう

- 新しい DogNode を作ってみよう
- 新しい Topic の種類を増やしてみよう
- Brain を組み合わせて賢いロボット犬にしよう
