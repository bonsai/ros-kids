# シラバス

## 科目名

**ロボット犬で学ぶ Java & ロボティクス**
（Java Programming through Robotics）

## 単位数

2 単位（講義 15 回 + 演習）

## 対象

情報科学科 / 総合科学科 1〜2 年次
Java 未経験可。プログラミング基礎を履修していることが望ましい。

## 授業計画

| 回 | 課 | タイトル | 内容 |
|----|-----|----------|------|
| 1 | 0 | Java にようこそ | 環境構築、Hello World、変数と型 |
| 2 | 1 | クラスとオブジェクト | クラス定義、インスタンス化、メソッド呼び出し |
| 3 | 1 | カプセル化 | private / public、getter / setter、データ保護 |
| 4 | 1 | コンストラクタ | 初期化、this、オーバーロード |
| 5 | 2 | 継承 | extends、super、メソッドオーバーライド |
| 6 | 2 | インターフェース | interface、implements、設計の抽象化 |
| 7 | 3 | 多態性（ポリモーフィズム） | アップキャスト、動的ディスパッチ |
| 8 | 3 | 戦略パターン | 頭脳差し替え、依存性注入 |
| 9 | 4 | ロボットの頭脳 | Brain インターフェース、AI エージェント |
| 10 | 4 | 世界モデル | State / Action、予測と意思決定 |
| 11 | 5 | pub/sub 通信 | トピック、購読、ブロードキャスト |
| 12 | 5 | マルチエージェント | 複数ノードの協調、同期 |
| 13 | ― | 総合演習 | ロボット犬シミュレータの完成 |
| 14 | ― | 発表 | 自由課題の発表 |
| 15 | ― | まとめ | 総復習と発展 |

## 評価基準

- 演習問題（各課末）: 40%
- 総合演習: 30%
- 自由課題発表: 30%

## 使用環境

- Java 21
- 任意の Java IDE（Eclipse / IntelliJ IDEA / VS Code）
- ブラウザ上の Java プレイグラウンド（TeaVM）
