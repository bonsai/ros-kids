# 用語集

| 用語 | 英語 | 意味 | 登場課 |
|------|------|------|--------|
| クラス | class | オブジェクトの設計図。プロパティとメソッドを定義する | 1 |
| オブジェクト | object | クラスから作られた実際のデータのかたまり（インスタンス） | 1 |
| インスタンス | instance | クラスから生成されたオブジェクトのこと | 1 |
| インスタンス化 | instantiation | `new` を使ってオブジェクトを作ること | 1 |
| プロパティ | property | オブジェクトが持つデータ（属性） | 1 |
| メソッド | method | オブジェクトが持つ処理（関数） | 1 |
| カプセル化 | encapsulation | データを隠して必要な操作だけ公開する設計思想 | 2 |
| アクセス修飾子 | access modifier | アクセス範囲を制御するキーワード（private, public など） | 2 |
| private | private | 同じクラスの中からしかアクセスできない | 2 |
| public | public | どこからでもアクセスできる | 2 |
| protected | protected | 同じパッケージと子クラスからアクセスできる | 4 |
| getter | getter | プロパティの値を取得するメソッド | 2 |
| setter | setter | プロパティの値を設定するメソッド | 2 |
| バリデーション | validation | 値が正しい範囲かどうかをチェックすること | 2 |
| コンストラクタ | constructor | オブジェクト生成時に自動実行されるメソッド | 3 |
| this | this | 自分自身のオブジェクトを指すキーワード | 3 |
| オーバーロード | overload | 同じ名前で引数の異なるメソッドを複数定義すること | 3 |
| 継承 | inheritance | 既存のクラスの機能を引き継いで新しいクラスを作ること | 4 |
| extends | extends | 継承を宣言するキーワード | 4 |
| スーパークラス | superclass | 親クラス（継承の元になるクラス） | 4 |
| サブクラス | subclass | 子クラス（継承して作られたクラス） | 4 |
| オーバーライド | override | 親クラスのメソッドを子クラスで上書きすること | 4 |
| super | super | 親クラスを指すキーワード | 4 |
| インターフェース | interface | メソッドのシグネチャだけを定義した抽象的な型 | 5 |
| implements | implements | インターフェースを実装することを宣言するキーワード | 5 |
| 抽象メソッド | abstract method | 中身（実装）がないメソッド | 5 |
| 多態性 | polymorphism | 同じ操作でもオブジェクトによって振る舞いが変わる仕組み | 6 |
| アップキャスト | upcast | 子クラスの型を親クラスの型に変換すること | 6 |
| 動的ディスパッチ | dynamic dispatch | 実行時にどのメソッドが呼ばれるかが決まる仕組み | 6 |
| 戦略パターン | strategy pattern | アルゴリズムをクラス化して差し替え可能にするデザインパターン | 7 |
| 依存性注入 | dependency injection | 外部から依存オブジェクトを受け取る設計手法 | 7 |
| コンポジション | composition | オブジェクトを部品として組み合わせること | 7 |
| AI エージェント | AI agent | 環境を認識し、意思決定して行動するプログラム | 8 |
| 状態 | state | ある時点におけるシステムの状況 | 8 |
| 行動 | action | エージェントが実行する動作 | 8 |
| 世界モデル | world model | 行動が状態をどう変えるかを予測するモデル | 9 |
| ROS | Robot Operating System | ロボットソフトウェア開発のためのフレームワーク | 10 |
| ノード | node | ROS における独立した機能モジュール | 10 |
| トピック | topic | ノード間でメッセージをやりとりする通信路 | 10 |
| publish | publish | トピックにメッセージを送信すること | 10 |
| subscribe | subscribe | トピックからメッセージを受信すること | 10 |
| 疎結合 | loose coupling | モジュール間の依存関係が弱い設計 | 10 |
| マルチエージェント | multi-agent | 複数のエージェントが協調して動作するシステム | 11 |
| イベント駆動 | event-driven | イベントの発生に応じて処理を実行する方式 | 11 |
